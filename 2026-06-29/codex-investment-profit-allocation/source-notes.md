# 资料边界

## 原始素材

- 用户提供的“投资赚的钱属于偏财，必须得花出去吗？”长文，包含正财/偏财解释、偏财流通逻辑、三类用法和两个误区。

## 改写边界

- 本文是观点改写，结合 Codex 工作流表达，不是命理咨询、投资建议或财务规划服务。
- 文中的 `30% / 40% / 30%` 只是示例分流框架，不适用于所有人。
- “正财/偏财/身弱/比劫”等属于传统命理表达，文章已尽量转译为现实里的现金流、风险承受力、情绪稳定性、外部关系干扰等概念。
- Codex 在文中承担“记录、分类、复盘、提醒风险”的执行角色，不承担买卖判断、荐股、收益预测或自动交易角色。

## 合规提醒

- 不承诺投资收益。
- 不鼓励高风险投机、加杠杆或自动交易。
- 发布时建议保留“不构成投资建议”的边界表达。
