# 配图使用说明

## 图片清单

1. `assets/01-title-5x2.png`
   - 用途：X 文章标题图 / 封面图
   - 尺寸：2500 x 1000
   - 比例：5:2
   - 建议作为第一张图

2. `assets/02-three-way-split.png`
   - 用途：正文插图 1
   - 建议位置：放在“三类分流”段落附近
   - 作用：解释流通、固化、再投入三类资金去向

3. `assets/03-codex-workflow.png`
   - 用途：正文插图 2
   - 建议位置：放在“这就是 Codex 可以介入的地方”附近
   - 作用：展示 Codex 记录、分类、分配、复盘的执行链路

4. `assets/04-boundary.png`
   - 用途：正文插图 3
   - 建议位置：放在“不要让 Codex 替你判断买卖点”附近
   - 作用：明确 Codex 不是荐股工具，而是复盘执行环境

## 优先发布组合

如果只发 2 张图：

1. `01-title-5x2.png`
2. `02-three-way-split.png`

如果发完整图组：

1. `01-title-5x2.png`
2. `02-three-way-split.png`
3. `03-codex-workflow.png`
4. `04-boundary.png`
