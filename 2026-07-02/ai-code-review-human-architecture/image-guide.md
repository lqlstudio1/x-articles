# 配图说明

## 封面锚点

- 模板类型：冲突型 + 架构蓝图型。
- 一眼承诺：AI 代码不是不用审，而是人不要审每一行，要审架构。
- 主视觉锚点：左侧超大标题 `别审每行`，右侧开发者桌面、代码窗口和架构蓝图。
- 锚点位置：左侧 54% 放主标题和副标题，右侧 46% 放场景锚点。
- 文字层级：主标题最大，副标题次级，底部标签最小。
- 缩略图检查：80px 宽缩略图里应能看清 `别审每行` 与 `审架构`。

## 风格

- 本篇不继续使用高饱和红黑封面，改用深蓝工业蓝图 + 暖橙强调色。
- 每张图都保留一个场景锚点：开发者桌面、代码窗口、架构图、文档面板。
- 图片文字控制在少量大字，不堆正文。
- 不伪造真实 IDE、GitHub、OpenAI、X 或任何平台界面；所有界面均为抽象示意。
