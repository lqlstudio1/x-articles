# 资料边界

## 原始素材

- 用户提供的 X 收益化付款前账号功能被暂停的风险提醒，以及一组账号健康实践建议。

## 官方资料核对

本次使用 X 官方 Help Center 的 Authenticity 页面做边界核对：

- URL：`https://help.x.com/en/rules-and-policies/authenticity`
- 页面日期标注：April 2025。
- 相关要点：
  - X 不允许通过不真实账号、行为或内容操纵平台或干扰服务。
  - 内容垃圾信息包括批量、重复、无关或未经请求的内容，以及与原帖主题无关的推广式回复。
  - 互动垃圾信息包括交换互动、指标膨胀、关注后取关、大量无差别关注/取关、自动化或激进互动等。
  - 违规后可能出现触达限制、临时功能限制、账号锁定或暂停等执行措施。

## 写作边界

- 本文不是 X 官方规则翻译，也不是收益化资格说明。
- 原文包含具体频率数字，升级版中没有把任何次数写成“安全阈值”。
- 本文不提供规避平台风控的方法，重点是账号健康、真实互动、原创素材和减少平台操纵风险。
- 文中提到 Codex 的用途仅限复盘、记录、整理素材和检查风险，不用于批量互动。

## 合规提醒

- 不鼓励互关互赞、批量关注取关、批量评论、批量点赞转推。
- 不建议使用互动暗号或复制粘贴评论。
- 不建议重复搬运他人图片、视频或跨平台旧素材。
- 不建议使用自动化工具刷互动。
