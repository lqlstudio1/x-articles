# 资料边界

这篇文章没有把“算法喜欢什么”写成确定公式，只使用公开资料能确认的方向：

- X 的 For You 时间线会展示关注账号、Topics 以及推荐内容；推荐内容会参考流行度、网络互动等信号，并强调相关、可信、安全。
- X 的推荐系统说明提到推荐基于多种信号，没有单一信号被静态赋予最高权重。
- X 的真实性政策反对批量、重复、无关、高频回复、操纵互动、互粉互赞等行为。
- X Analytics / Post Activity Dashboard 可查看展示、互动、互动率、回复、转发、关注等数据，用于复盘内容表现。
